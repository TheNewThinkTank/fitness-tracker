src.one_rep_max module

acsm_1rm, epley or brzycki formulas are used to implement the 1RM estimation.

ACSM 1RM formula

\[\frac{w}{\frac{100 - r \cdot 2.5}{100}}\]

Epley 1RM formula

\[w \cdot \frac{1 + r}{30}\]

Brzycki 1RM formula

\[w \cdot \frac{36}{37 - r}\]
